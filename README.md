# Poly Randomizer

A static site that displays random polymorphic siteswap juggling patterns animated via [JugglingLab](https://jugglinglab.org).

## GitHub Pages setup

1. Push this repository to GitHub.
2. Go to **Settings → Pages**.
3. Under **Source**, select **Deploy from a branch**, choose `main`, folder `/` (root), and save.
4. The site will be available at `https://<username>.github.io/<repo-name>/`.

## Regenerating data

Pattern data in `data/` was generated by:

```
ruby generate_data.rb
```

Requires Ruby 3+.
